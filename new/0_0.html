Polygon Mesh Cutting Tool

Initially, I began by dividing the scalp for tube grooming. My first attempt involved utilizing Topobuild to generate a scalp mesh atop the base scalp mesh for retopologizing purposes. However, I encountered an obstacle in creating split lines for the hair on the human scalp using this method.

Consequently, I developed a tool in Houdini to split the mesh based on mouse input, allowing for drawing directly on the mesh. This enabled me to split the scalp mesh according to the reference plan and to create a more detailed mesh for the hair tube model.
