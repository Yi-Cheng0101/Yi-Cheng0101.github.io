Procedural Object Look Development

The goal of this project was to create realistic materials using a procedural Houdini workflow. I found a detailed bag on a second-hand seller website, showcasing signs of aging and use in real life. I was intrigued by these surface imperfections and started to recreate the backpack with meticulous attention to detail, aiming for a 1:1 replication.

Thread simulation

To begin, I generated a twisted rope using Houdini and produced a displacement map to accentuate the thread details. Using procedural modeling workflow, I textured the rope’s threads and refined their natural shape using Houdini’s Vellum solver. To achieve a bell-shaped appearance for the rope’s threads, I applied finer Vellum constraints to the top portion of the threads before running the Vellum simulation.
